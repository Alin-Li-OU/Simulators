# Alin Li Simulators

> Collection of open-source educational simulators by **Alin Li OÜ** (Estonia).
> Each simulator is a self-contained single-file HTML document.
> No installation, no build step, no external runtime dependencies.

**Live at:** <https://alinli.com/simulators.html>

## Simulators

| Simulator | Version | Description |
|---|---|---|
| [Alin Li Computer Explorer](./computer-explorer.html) | v1.0.0 | An interactive boot-up journey: follow what happens inside a computer when you press the power button. |
| [Alin Li Git Simulator](./git-simulator.html) | v1.0.0 | Learn Git interactively, the fun way. |
| [Alin Li Linux Explorer](./linux-explorer.html) | v1.0.0 | Practise Linux commands hands-on in your browser, with no installation. |
| [Alin Li NiFi Simulator](./nifi-simulator.html) | v1.0.0 | Visualise Apache NiFi data flows with mock infrastructure: files, tables, and APIs. |

## Open-source compliance

Our open-source compliance programme is **self-certified to
ISO/IEC 5230:2020** (OpenChain Specification 2.1).

**Current posture:** the simulators bundle no third-party open-source
components at runtime. They use operating-system fonts only.

| Document | Purpose |
|---|---|
| [`NOTICE`](./NOTICE) | Top-level attribution. |
| [`OSS-REVIEW.md`](./OSS-REVIEW.md) | Per-component register (currently nil). |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | OSS policy for contributors. |
| `credits/<slug>.html` | End-user attribution page for each simulator. |
| `sbom/<slug>.cdx.json` | CycloneDX 1.5 SBOM for each simulator. |
| `oss/index.html` | Public OSS landing (mirrored at <https://alinli.com/oss/>). |
| `oss/conformance.html` | Public ISO/IEC 5230:2020 conformance statement. |
| `oss/review-process.html` | Visual diagram of the bi-annual review cycle. |
| `oss/source-offer.html` | Source-offer statement (currently nil). |
